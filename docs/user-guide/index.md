# User Guide

Sec-Check is available in three forms. Choose the guide that matches your workflow:

| Guide | Best For |
|:------|:---------|
| [CLI Tool]({% link user-guide/cli.md %}) | Automated scanning from the terminal or CI/CD pipelines |
| [Copilot Toolkit]({% link user-guide/copilot-toolkit.md %}) | Interactive scanning inside GitHub Copilot Chat |
| [VS Code Extension]({% link user-guide/vscode-extension.md %}) | Native VS Code experience with dashboard and tree views |
